Build by typing

# make

Start Minesweeper with
./Minesweeper a b c ~
       
  a Level (1-100);
  b Row/Collumns of area;
  c Number of bombs in area.

  
 During the game, the following keys are availiable:

         8 ... up
         2 ... down
         4 ... left
         6 ... right
         5 ... mark bomb (where you think one is)
         Space ... open field
         q ... quit
